# hover-circle-mover

Create a web app that contains a circle 50 pixels in diameter. When a user mouses over the circle it moves to a random location 200 pixels away.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/hover-circle-mover.git
cd hover-circle-mover
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
